## 2025-02-14 - React.memo on Grid Cells
**Learning:** In React components rendering large grids (like a 35-day month view), mapping over an array in the parent component and rendering raw HTML elements directly causes all 35 cells to re-render whenever the parent's state changes (e.g., `currentDate` ticking forward, or selection state changing), even if the individual cell's data hasn't changed.
**Action:** Extract the individual cell rendering into its own React component wrapped in `React.memo()`. Then, cache expensive inner calculations like `getEventColorStyles` using `useMemo` within that cell component. Pass the specific sliced data (`dayEvents`) to each cell. This prevents full-grid re-renders and isolate re-renders to only the cells whose specific props (like `isToday` or `dayEvents`) have actually changed.
